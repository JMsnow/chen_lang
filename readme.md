# chen_lang
## 用 [Rust](https://www.rust-lang.org) 写的 一个小编程语言


```
sum = 0
i = 100
print(i)
for i {
    sum = sum + i
    i = i + -1
}
print(sum)
```
---
输出:
```
100
5050
```
